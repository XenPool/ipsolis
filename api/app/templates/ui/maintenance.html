How it works

  1. Copy your Install ID from the section above, then visit the ip·Solis Shop and paste it during checkout.
  2. Download the signed .lic file and upload it below. The signature is checked against the public key baked into this build — if the file is corrupt or from a different issuer, the upload is rejected and your current license stays untouched.
  3. On success, the new license takes effect immediately across all processes (api, worker, beat). No restart needed.
  4. All features are available regardless of license status. A valid commercial license is required for production use under the XenPool Commercial Source License.
